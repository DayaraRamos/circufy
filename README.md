# ⚡ Circufy — Simulador de Compuertas Lógicas

Aplicación web interactiva para aprender y practicar las 8 compuertas 
lógicas del Álgebra Booleana.

## ¿Qué puedes hacer?
- Completar tablas de verdad interactivas
- Armar circuitos lógicos con drag & drop
- Verificar tus respuestas en tiempo real

## Tecnologías
HTML · CSS · JavaScript
